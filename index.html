1. MV2 Depreciation Vulnerability

Description: The MV2 Depreciation Vulnerability is a vulnerability discovered by deacon. (codename=bishop) It exploits a vulnerability in which if a organization managed extension is below Version 3, the chrome system will disable it with admin privilages.

Impact:This exploit, depending on your district, can completely delete the lightspeed classroom and filter agent. allowing you to visit any website of your choice. and removing teachers access to your screen.

Steps to Execute:

  1. Open chrome://flags. If it's blocked, you're screwed.
  2. Search "extension manifest".
  3. You should see these three flags. Set them as I've typed:
    • Extension Manifest V2 Deprecation Warning Stage: Enabled
    • Extension Manifest V2 Deprecation Disabled Stage: Enabled
    • Changes accessibility features from extension manifest v2 to v3: Default
  4. Restart your Chromebook. And if your Lightspeed wasn't updated, it should be disabled. Some people still have classroom (what allows teachers to see your screen) and some don't. It depends on your district.
